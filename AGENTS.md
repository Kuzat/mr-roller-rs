# AGENTS.md

## Human-authored game text

When implementing new game content or features, do not invent user-facing creative text without asking first.

This applies to, but is not limited to:

- Item names
- Item descriptions
- Flavor text
- Event names
- Event descriptions
- Character, location, monster, encounter, quest, or ability names
- Any other text that affects the feel, tone, or fiction of the game

If a change needs this kind of text, pause and ask the user/project owner to provide it. Use the exact wording they give unless they explicitly ask you to edit or propose alternatives.

Do not fill in placeholder names, temporary flavor text, or final copy yourself unless the user has specifically asked you to do so. If code needs a temporary value to compile, ask for the value first or clearly request permission to use a technical placeholder.
